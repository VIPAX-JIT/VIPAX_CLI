# Vipax CLI

A simple command-line developer toolkit built with **Node.js** and **TypeScript** as a university project.

Uses **Commander.js** for commands, **Axios** for API calls, and **Chalk** for coloured output.

---

## Project Structure

```
VIPAX_CLI/
├── src/
│   └── vipax.ts    ← Your source code (TypeScript)
├── dist/
│   └── vipax.js    ← Auto-generated on build (do not edit)
├── package.json
├── tsconfig.json
└── README.md
```

> **Note:** Only ever edit files inside `src/`. The `dist/` folder is auto-generated by TypeScript and is what Node.js actually runs.

---

## Installation & Setup

```bash
# 1. Install dependencies
npm install

# 2. Compile TypeScript → JavaScript
npm run build

# 3. Link the CLI globally (run once)
npm link
```

After `npm link` you can use the `vipax` command from anywhere in your terminal.

---

## Usage

```bash
# Run directly
node dist/vipax.js <command>

# Or after linking globally
vipax <command>
```

## Commands

| Command | Description | Example |
|---|---|---|
| `greet [name]` | Greet a user by name | `vipax greet Jatin` |
| `add <n1> <n2>` | Add two numbers | `vipax add 10 5` |
| `subtract <n1> <n2>` | Subtract two numbers | `vipax subtract 10 3` |
| `multiply <n1> <n2>` | Multiply two numbers | `vipax multiply 4 3` |
| `divide <n1> <n2>` | Divide two numbers | `vipax divide 22 7` |
| `joke` | Fetch a random joke | `vipax joke` |
| `github <username>` | Fetch GitHub user profile | `vipax github torvalds` |
| `quote` | Fetch a motivational quote | `vipax quote` |
| `coinflip` | Flip a coin (Heads / Tails) | `vipax coinflip` |
| `roll [sides]` | Roll a dice (default: 6) | `vipax roll 20` |
| `fileinfo <file>` | Show file metadata | `vipax fileinfo package.json` |

## APIs Used

| API | Endpoint | Command |
|---|---|---|
| Official Joke API | `https://official-joke-api.appspot.com/random_joke` | `joke` |
| GitHub REST API | `https://api.github.com/users/<username>` | `github` |
| DummyJSON Quotes | `https://dummyjson.com/quotes/random` | `quote` |

---

## After Making Code Changes

Whenever you edit `src/vipax.ts`, re-run the build:

```bash
npm run build
```

Then run your command as usual.

---

## Tech Stack

- **Node.js** — Runtime
- **TypeScript** — Typed JavaScript (source in `src/`)
- **Commander.js** — CLI command parsing
- **Axios** — HTTP API requests
- **Chalk** — Coloured terminal output
